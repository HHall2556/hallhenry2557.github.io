Project Requirements

Project Overview

  1. Describe your application and its purpose

    • Frequently there is one person that makes the trip to the store in a household at a time and tracking the needs of all the members of a household can be tricky. Certain items are best purchased at certain stores or only found at one store while others are commodities and knowing the price vs other places you have purchased it can be important. Furthermore, frequently the person shopping needs to communicate with the person at home about the product to further refine it. Another issue with a cloud based application is that Sometimes connectivity inside the store can be limited, so the ability to cache the list is important.

  2. Intended users of the Website

    • Members of any household who want to coordinate shopping and purchasing groceries.

  3. Overview of the content of the website

    1. Main page that introduces project and the features
    2. Features
    3. Partnership page
    4. About page (history as developer) + contact
    5. Project Progress
