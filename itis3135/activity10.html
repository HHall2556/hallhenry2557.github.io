Notes

I had many difficulties with this project. My main problem was with the implementation of the callback function. Originally, I had made a new function that can be called from anywhere that tried to switch the caption and img. This was bad. Once I changed it to an internal function, and split the caption and image properly, I got the desired result.

On my local Server, everything works properly. On charlotte.edu, it changes the image late. Not sure if it is the server or if I did not preload the images properly.

Help

I got help from numerous sources, but I will only put the most helpful here.

  • StackOverFlow.com: This page is a simple example for changing one image to another that made me realize I was massively overcomplicating everything and allowed me to make a workable solution.
  • StackOverFlow.com: Shows how to prevent default behaviour of anchor tags.
  • StackOverFlow.com: Example about preloading. I changed mine to preload in a dynamic fashion, but in the future I would do it more similar to example.
