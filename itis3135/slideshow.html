These are the ways mine differ from the source

  • I put all my images in figures for figcaptions

    • Images and figures don't play nice together, so I had to change the layout
  • I have different sized images
  • I changed the functionality to cycle in a circle instead of stopping once you get to the end of the list
  • Not sure if result of different browser or version, but his code needed to be changed to resolve minor errors
    • In his, img.active selected the first image, but I had to hard code an 'active' class to the first element
